# LBW-Documents
## 気象業務支援センターからのデータ受信について

[気象業務支援センター（JMBSC）](https://www.jmbsc.or.jp/jp/online/online.html) からは、主に以下の2種類の形式でデータを受信しています。

### 電文形式データ

受信経路は以下の通りです。

| 受信経路 | 受信場所 | 受信ソフトウェア |
|---|---|---|
| 専用回線（NTT ビジネスイーサ） | ハウジング内サーバー | 独自プロトコル。富士通提供ライブラリ＋カスタムアプリ |

### ファイル形式データ

受信経路は以下の3つに分かれています。

| 受信経路 | 受信場所 | 受信ソフトウェア |
|---|---|---|
| 専用回線（NTT ビジネスイーサ） | ハウジング内サーバー | ftpサーバー(vsftp) |
| インターネット経由 | ハウジング内サーバー | ftpサーバー(vsftp) |
| インターネット経由 | VPS | ftpサーバー(vsftp) |

## メモ

- 電文形式の独自プロトコルは ftpサーバーでの代替を検討中。
- ハウジングの縮小・廃止に向けてクラウド/VPSへの代替を検討中。
