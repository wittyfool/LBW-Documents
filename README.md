# LBW-Documents
## 気象業務支援センターからのデータ受信について

[気象業務支援センター（JMBSC）](https://www.jmbsc.or.jp/jp/online/online.html) からは、主に以下の2種類の形式でデータを受信しています。

### 電文形式データ

経路は以下の通りです。

| 経路 | 設備 |
|---|---|
| 専用回線（NTT ビジネスイーサ） | ハウジング内サーバー |

受信ソフトウェア: 独自プロトコル。富士通提供ライブラリ＋カスタムアプリ

### ファイル形式データ

経路は以下の3つに分かれています。

| 経路 | 設備 |
|---|---|
| 専用回線（NTT ビジネスイーサ） | ハウジング内サーバー |
| インターネット経由 | ハウジング内サーバー |
| インターネット経由 | VPS |

受信ソフトウェア: ftpサーバー(vsftp)

## メモ

- 電文形式の独自プロトコルは ftpサーバーでの代替を検討中。
- ハウジングの縮小・廃止に向けてクラウド/VPSへの代替を検討中。
